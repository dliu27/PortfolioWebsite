Using NPM:

```bash
# Switch to proper node version
nvm use 13.14.0

# Install dependencies
$ npm install

# Start development server
$ npm run develop
```

Using Yarn:

```bash
# Install dependencies
$ yarn

# Start development server
$ yarn develop
```

Local server: `http://localhost:8000/`
